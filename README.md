# VAT Microsimulation – Côte d’Ivoire

This project develops a microsimulation model to estimate the incidence of VAT using household survey data (EHCVM).

## Objective
To assess the distributional impact of VAT across households and income groups in Côte d’Ivoire.

## Methodology
The model follows a CEQ-type framework:
- Household consumption data (EHCVM)
- Product-level VAT rates
- Computation of effective tax burden at the household level

## Project Structure
- `data/` : raw and cleaned datasets
- `code/` : Stata scripts for data processing and simulation
- `output/` : tables and figures
- `doc/` : methodological notes
